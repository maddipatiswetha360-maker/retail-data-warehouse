# Project Explanation

## Objective
To design and build a scalable data warehouse for retail sales analysis.

---

## Workflow

1. Created dimension tables and fact table
2. Loaded raw data into staging table
3. Applied transformations using joins
4. Loaded data into fact table
5. Generated business insights using SQL queries

---

## Concepts Used
- Star Schema Design
- ETL (Extract, Transform, Load)
- Joins (Inner Join)
- Aggregations (SUM, GROUP BY)
- Data Quality Checks

---

## Outcome
- Generated insights like:
  - Top products by revenue
  - Monthly sales trends
  - Customer spending patterns

---

## Conclusion
This project demonstrates strong SQL skills and understanding of data warehousing concepts, suitable for real-world data engineering scenarios.
